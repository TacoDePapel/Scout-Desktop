What Scout does

🎙️
Record any task

Capture your screen + voice. Scout's AI writes a clean step-by-step skill guide automatically.

⚙️
Run real tasks for you

Tell Scout what you want done. The built-in Claude agent uses your terminal, files, apps, and browser to finish the job — autonomously.

👁️
Watches your screen

Optional passive monitor keeps the last 5 minutes of your desktop locally so you can turn anything interesting into a skill in one click.

🔌
MCP-connected

Auto-loads any MCP server in your ~/.claude.json — Slack, Notion, GitHub, Linear, you name it.
